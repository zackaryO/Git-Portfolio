BigToa — AI Home Inventory

Insurance · Estate planning · Collections

Full-stack SaaS that uses Anthropic Claude to extract structured data from photos of personal belongings (make, model, serial, value), then organizes everything for insurance claims, estate planning, and collection tracking. Includes a zero-knowledge AES-256-GCM digital vault, two-trustee emergency access, eBay / Facebook Marketplace integration, Stripe subscriptions, passkey login, and SOC-2-style audit logging.

  • Next.js 16
  • TypeScript
  • Chakra UI v3
  • Django 5 / DRF
  • PostgreSQL
  • Redis
  • Celery
  • Stripe
  • Claude AI
  • AWS S3 / CloudFront
  • Render & Vercel

Wheel Alignment Simulator

Three.js · Teaching aid for automotive technicians

An interactive 3D playground for understanding caster, camber, toe, and SAI. Drag a slider and watch the wheel tilt; steer the car and watch the body lift. Visualizes spindle / steering / toe axes with real-time geometry tracers and a conceptual-vs-actual mode for teaching alignment math.

  • Three.js
  • Angular
  • TypeScript
  • WebGL
  • GitHub Pages
